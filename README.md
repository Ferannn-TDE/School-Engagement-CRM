# School-Engagement-CRM


## SchoolFacilityUploader

### Conda Environment Setup

Build and activate the environment:

    ```bash
    conda env create -f environment.yml
    conda activate SeniorProject
    ```

### pip Requirements Setup

If you prefer using pip:

    ```bash
    pip install -r requirements.txt
    ```

### Running the Script

After setting up your environment, run:

```bash
python SchoolFacilityUploader.py
```

### Notes

- Make sure your `.env` file with Supabase credentials is in the project directory.
